## Description

What does it mean to inherit systems you didn't build? I was honored to present on the hidden work behind trusting data, tools, and systems at Northern Illinois University. This session examines the procedural habits required to manage data and systems that aren't our own. We will break down strategies for maintaining high performance when working with everything from unfamiliar tabular data to the black-box nature of large language models.
